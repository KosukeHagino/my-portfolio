About

萩野 弘祐 KOSUKE HAGINO

これまで設計士として、現場の使いやすさと効率的な作業動線を追求した厨房の設計に携わってきました。「ものづくりで人を喜ばせること」にやりがいを感じる中、怪我による療養を経験。それを機に、以前より興味のあったWeb制作の世界への転身を決意しました。

設計士として培った「ユーザー視点」と「最適な動線を考える力」は、WebのUI/UXデザインにも通じると考えています。単に形を作るだけでなく、ユーザーの意図を汲み取り、課題を解決できるフロントエンドエンジニアを目指します。

Skills

  • デザイン DESIGN

    Webサイトやバナーを制作することができます。デザイン4原則に基づいた、見やすく整理されたレイアウトを基本とし、そこに厨房設計で培った「ユーザー視点」を掛け合わせ、意図が正しく伝わる使い勝手の良いデザインを心がけています。

    Figma Photoshop Illustrator

  • コーディング CODING

    厨房設計で培った設計図を正確に読み取り形にする力を活かし、デザインカンプを忠実に再現します。また、後々のメンテナンスを考えて、図面を引くように規則性のある整ったコードを書くことを大切にしています。レスポンシブ対応や、JavaScript/jQueryによる動きの実装、PHP/SQLを用いたお問い合わせフォームの構築まで、使う人に優しい実装を心がけています。

    HTML CSS SCSS JavaScript jQuery PHP WordPress SQL

開発環境 DEVELOPMENT ENVIRONMENT

個人開発ではSourcetreeを使用してGitでのバージョン管理と、GitHubでチーム内での作業連携やコードの共有、承認制などを常に意識しながら管理しています。コーディング作業はVS Code、PHPやDBを用いた開発ではローカル環境としてXAMPPを使用しています。

Strengths

  • 調整力

    建築図面と実際の現場寸法の差異を、可変寸法の機器を使って現場で調整してきた経験をコーディングに活かします。デザイン案をただ数値通りに組むのではなく、実機（スマートフォンやPC）での見え方の違いを考慮し、最終的に使う人が違和感なく操作できるよう、細部まで収まりを整える実装を大切にしています。

  • 提案力

    厨房設計では、調理工程や人・器具・食器の流れを深く理解しなければ、使い勝手の良い空間は作れません。この経験を活かし、コーディングに入る前に「誰がどう使うか」という動線を徹底して確認します。事前の丁寧な対話で認識を合わせることで、制作のズレを最小限に抑えます。

  • 共有力

    厨房設計の際、施主様だけでなく施工に関わる多くの業者様が目にすることを意識し、「誰が見ても分かりやすい正確な図面」を作ることを心がけてきました。Web制作でもこの姿勢を貫き、チームの誰もが意図を正確に把握できる整理されたコードを書くこと、そして円滑なコミュニケーションを常に意識しています。
